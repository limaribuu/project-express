Book API - Express JS

1. Tujuan Project

Project ini bertujuan untuk membuat REST API sederhana untuk mengelola data buku menggunakan Express JS.
Fitur utama:
- Mengambil daftar semua buku
- Mengambil detail buku berdasarkan ID
- Menambah buku baru
- Mengupdate data buku
- Menghapus buku
